# 👤 Random Username Generator (Python CLI)

A simple Python command-line tool that generates random usernames using combinations of adjectives, nouns, and numbers.

---

# 📌 Features

* Generates random usernames
* Combines adjectives + nouns + numbers
* User chooses how many usernames to generate
* Beginner-friendly Python project

---

# 📂 Project Structure

```
random_username_generator
│
├── username_generator.py
├── README.md
```

---

# ⚙ Requirements

* Python 3.x
* No external libraries required

---

# 🚀 Run the Program

Navigate to the project folder:

```bash
cd random_username_generator
```

Run the script:

```bash
python username_generator.py
```
